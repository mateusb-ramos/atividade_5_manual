# O que é JavaScript
JavaScript é uma linguagem de programação criada para trazer "vida" a websites, hoje usada para deixar as páginas interativas e dinâmicas, ela roda em navegadores de internet e pode integrar e facilitar a comunicação dentro de um website. Além disso, pode ser usado sem estar conectado a internet. 

# Relação de JavaScript com HTML e CSS
JavaScript, HTML e CSS são fundamentais para o desenvolvimento web. O HTML é uma linguagem de marcação, usado para criar a estrutura da página, usando tags para sinalizar onde cada tipo de elemento será implementado, o CSS é uma linguagem de estilos que separa a parte estética, da parte estrutural, controlando a apresentação, o layout e o design de websites, já o JavaScript, serve para tornar os websites interativos e reativos, permitindo o processamento e transformação de dados enviados e recebidos. Sendo assim, eles se complementam para criar sites funcionais, atraentes e interativos.

# JavaScript escrito dentro da própria página HTML
Podemos incorporar JavaScript em um documento HTML usando a tag <script></script> dentro do <head> ou <body> ou podemos colocar link de arquivos JS externos usando a tag <script src="script.js"></script> e podemos também inserir códigos JS diretamente dentro de tags HTML usando o atributo onclick, onload, etc.
